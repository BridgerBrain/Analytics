README file for Jira API:

check_working_datetime.py - A file which checks if a given datetime is considered as working or not.

get_environmental_variables.py - A file which gives us the credentials to connect to Jira API.

get_issues.py - A file which access Jira issues and gives us all the required information about them.

main.py - The main file to run YouTube files and create the dataset.

jira_datatype.xlsx - A file which describes the datatype of each column of the dataset.

requirements.txt - A file which contains the libraries that are used for the project.
