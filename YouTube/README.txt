README file for YouTube API:

get_channel_statistics.py - A file which access channel's statistics using a given YouTube channel ID.

get_channel_video_statistics.py - A file which gives us video information and statistics of a YouTube channel.

get_playlist_aux_functions.py - An auxiliary file which contains functions related to YouTube playlists.

get_playlist_statistics.py - A file which gives us video information and statistics using a given YouTube playlist ID.

get_youtube_variables.py - A file which gives us the credentials to connect to YouTube API.

main.py - The main file to run YouTube files and create the datasets.

requirements.txt - A file which contains the libraries that are used for the project.
